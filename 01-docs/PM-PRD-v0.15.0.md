# PM-PRD v0.15.0 — 优化结果中文翻译显示

## 概述

UI 增强：优化结果若为英文，在结果区下方增加「中文翻译」项。仅展示用，不影响实际提示词。

## 背景

- 目标用户中，**非英文母语用户**占多数
- 优化结果通常是英文（如 `A majestic feline with glowing eyes...`）
- 用户希望"看懂优化结果"，但**不需要把中文发给图片生成平台**
- 当前 UI 只显示英文，用户只能复制后自己翻译

## 功能清单

| 功能 | 等级 | 工作量 |
|------|------|--------|
| F1 中文翻译自动生成与展示 | P1 | 中 |

## F1: 中文翻译显示

### 交互

```
[优化结果]
  A majestic feline with glowing eyes sitting on a velvet throne, golden hour lighting, ...
  [耗时 1234ms] [👍] [👎]

  ▼ 中文翻译 (仅供参考)      ← 新增
    一只威严的猫科动物，双眼闪烁，
    端坐在天鹅绒宝座上，金色时刻的
    光线，4K 高清...
    [📋 复制中文]
```

### 关键决策

| 决策 | 原因 |
|------|------|
| **前端翻译，不调 LLM** | 简单 prompt 翻译可前端正则/字典，不增加 LLM 成本 |
| **可选后端 LLM 翻译** | 复杂/创意性 prompt 用 LLM 翻译更准确 |
| **折叠展开** | 节省屏幕空间 |
| **独立复制按钮** | 方便中文用户复制 |
| **仅当原文为英文时显示** | 中文 prompt 自身就是中文，无需翻译 |
| **不替换优化结果** | 中文只展示不写入主框（避免误复制） |

### 实现策略（两层）

**Layer 1：本地关键词替换**（前端，无网络）
- 维护 200+ MJ 风格关键词中英对照表
- `a majestic cat` → `一只威严的猫`
- 优点：0 成本、0 延迟
- 缺点：句式不顺

**Layer 2：LLM 翻译**（后端，可选）
- 调用 OpenAI/讯飞做翻译
- 优点：自然通顺
- 缺点：增加 LLM 调用费用

**默认仅启用 Layer 1**（够用且免费）。Layer 2 留作未来。
