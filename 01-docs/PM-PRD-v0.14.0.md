# PM-PRD v0.14.0 — 产品使用说明文档

## 概述

项目 13 个迭代后功能已完整，但用户入口分散在 7 份文档中，新用户无从下手。需要一个 `MANUAL.md` 统一覆盖所有功能的使用说明。

## 背景

| 当前问题 | 影响 |
|---------|------|
| README 只列特性，不教使用 | 新用户不知道怎么用 |
| 功能分散在 7 份文档中 | 查找成本高 |
| 无步骤式教程 | 需要自行摸索 |
| 无截图/示例 | 不知道界面长什么样 |

## 产出

`docs/MANUAL.md` — 一份文档覆盖全部使用场景

### 受众
- **初级用户**：只想打开网页点按钮
- **中级用户**：想用 CLI 和 API
- **高级用户**：想知道架构和配置

### 结构

1. **快速开始**（5 分钟上手）
2. **Web 界面使用指南**
   - 2.1 Prompt 工作台（优化/分类/评估/扩写/批量/A/B 多版本）
   - 2.2 数据看板
   - 2.3 配置
3. **CLI 使用指南**
4. **API 使用指南**
5. **高级功能**
   - RAG 知识库
   - 反馈闭环
   - 缓存池
   - 风格注入
6. **部署指南**
   - Docker
   - 本地开发
7. **常见问题**

### 规格

| 项 | 值 |
|------|------|
| 文档格式 | Markdown |
| 语言 | 中文（用户母语） |
| 目标大小 | 3000-5000 字 |
| 截图数 | 文本描述为主 |
| 存放位置 | `docs/MANUAL.md` |
