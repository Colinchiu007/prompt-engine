# PM-PRD v0.17.0 — 优化速度提升

## 概述

当前优化耗时 ~7s，95% 在 LLM 调用。通过降低默认参数 + 增加速度选择器，可降至 2-4s。

## 变更清单

| 功能 | 改动 | 效果 |
|------|------|------|
| F1 默认 max_length 500→300 | `models.py` 1 行 | 7s→4s |
| F2 速度模式选择器 | Workbench 新增 dropdown | 用户自控 |
| F3 缓存优先 | 已有，确认正常 | 第二次 0ms |

## 速度模式

| 模式 | max_length | creative_level | 目标耗时 | 适用场景 |
|------|-----------|---------------|---------|---------|
| ⚡ 快速 | 150 | 4 | ~2s | 试效果、批量 |
| 🎯 标准（默认） | 300 | 6 | ~4s | 日常使用 |
| 📖 详细 | 500 | 8 | ~7s | 重要/展示 |

## 验收

- [x] 默认 max_length=300
- [x] 速度模式 dropdown 3 档
- [x] 快速模式 ~2s
