# PM-PRD: Prompt Engine 全功能看板

## 概述

为 prompt-engine 添加 Web 图形界面，包含配置管理、数据看板、监控功能。

## 功能清单

| 模块 | 功能 | 说明 |
|------|------|------|
| **配置面板** | 平台策略管理 | 查看/编辑 7 个平台的策略配置 |
| | 分类器管理 | 查看分类器权重、通配符池状态 |
| | 供应商配置 | LLM 供应商切换（OpenAI/讯飞/Gemini） |
| | DSL 通配符管理 | 查看/编辑通配符池 |
| **数据看板** | 调用统计 | 请求量、成功率、平均耗时（时间线图） |
| | 分类分布 | 25 维风格分类使用比例（饼图/柱状图） |
| | 平台分布 | 各平台调用量（柱状图） |
| | 反馈统计 | 用户反馈数量、正确率趋势 |
| **Prompt 工作台** | 输入/优化 | 输入描述，选平台，查看优化结果 |
| | 风格分类 | 自动分类+手动修正 |
| | 评估对比 | 原始 vs 优化后 prompt 评估 |
| | 批量优化 | 批量处理+Excel 导出 |
| **监控** | 实时日志 | 请求日志流 |
| | LLM 调用统计 | 各供应商调用量、成功率、延迟 |

## 技术选型

| 层级 | 方案 | 理由 |
|------|------|------|
| 前端 | Vue 3 + Element Plus | 与 PROJECT-003 一致 |
| 图表 | ECharts | 调用趋势/分类分布/平台分布 |
| 后端 | FastAPI（已有） → 新增 `static/` + `templates/` | 复用现有 REST API |
| 部署 | FastAPI 内嵌静态文件 | 单文件部署，无需额外服务 |
